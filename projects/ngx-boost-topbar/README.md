# ngx-boost-topbar

Topbar component for Angular + Bootstrap 5 with Bootstrap Icons.

## Status

This library is a work in progress. The API and behavior are not yet stable.

## Development

```bash
# Build
ng build ngx-boost-topbar

# Test
ng test ngx-boost-topbar

# Lint
ng lint ngx-boost-topbar
```
